# Wooldridge Python Companion

Applied econometrics projects built alongside *Introductory Econometrics* (Wooldridge, 7th ed.), implemented in Python.

---

## Stack

`statsmodels` · `linearmodels` · `pandas` · `numpy` · `matplotlib` · `seaborn` · `wooldridge`

## Data

Data from the `wooldridge` Python package (`pip install wooldridge`), which includes all textbook datasets out of the box.
